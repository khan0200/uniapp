## üìû Input Format Reference

### Phone Number
```
Format: 00-000-00-00
Example: 99-123-45-67
```

### Passport Number
```
Format: AA0000000 (2 letters + 7 digits)
Examples: 
- AB1234567
- AC9876543
- AD1111111
```

### Full Name
```
Format: SURNAME, FIRSTNAME (all uppercase)
Examples:
- SMITH, JOHN
- LEE, SARAH
- JOHNSON, MICHAEL JAMES
```

### Address
```
Format: Full address in UPPERCASE
Example: 123 GANGNAM-GU, SEOUL, SOUTH KOREA
```

### Birthday
```
Range: January 1, 1980 - December 31, 2010
Format: YYYY-MM-DD
Example: 1998-05-15
```
